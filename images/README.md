# 游戏图片文件说明

## 图片准备

请将以下图片文件放入此文件夹中，以确保游戏中的图片显示功能正常工作：

### 开场场景图片
- intro_image1.jpg - 重要线索图片
- intro_image2.jpg - 神秘地图图片

### 勇敢路径场景图片
- brave_path_image1.jpg - 危险前方图片
- brave_path_image2.jpg - 神秘洞穴图片

### 谨慎路径场景图片
- cautious_path_image1.jpg - 隐藏线索图片
- cautious_path_image2.jpg - 地图标记图片

### 结局场景图片
- explore_alone_image.jpg - 独自探索的发现
- seek_help_image.jpg - 团队的力量
- gather_info_image.jpg - 关键情报
- make_plan_image.jpg - 完美计划

## 图片要求

- 建议使用JPG或PNG格式的图片
- 推荐分辨率：1280x720或更高
- 图片大小建议不超过2MB，以确保加载速度

## 自定义图片

如果您想自定义游戏中显示的图片，只需将您的图片文件以上述文件名保存在此文件夹中即可。您也可以在`js/game.js`文件中修改图片的路径、显示时间和持续时间。
