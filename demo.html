交互式剧情游戏 - 使用演示

项目概述

这是一个基于网页的交互式剧情游戏系统，允许用户通过选择不同的剧情分支来影响故事的发展方向。系统使用HTML5视频播放功能，在关键时刻暂停视频并提供选择，根据用户的选择导向不同的视频片段和结局。

立即体验游戏

使用步骤

1 准备视频文件

在项目的videos文件夹中准备以下MP4格式视频文件：

  • intro.mp4 - 游戏介绍和初始场景
  • brave_path.mp4 - 选择勇敢前进后的场景
  • cautious_path.mp4 - 选择谨慎撤退后的场景
  • explore_alone.mp4 - 独自探索结局
  • seek_help.mp4 - 寻求帮助结局
  • gather_info.mp4 - 收集信息结局
  • make_plan.mp4 - 制定计划结局

如果暂时没有视频文件，可以使用系统提供的"使用占位图像测试"功能。

2 启动游戏

打开index.html文件，点击"开始游戏"按钮开始体验。

3 做出选择

在视频播放到关键时刻时，系统会暂停视频并显示选项。点击你想要的选项，故事将根据你的选择继续发展。

4 探索多个结局

游戏有多个结局，你可以通过不同的选择路径来探索所有可能的结局。游戏结束后，点击"重新开始"按钮可以再次体验。

自定义游戏内容

如果你想自定义游戏内容，可以编辑js/game.js文件中的gameScenes对象：

const gameScenes = {
    'scene_id': {
        videoSrc: './videos/your_video.mp4',  // 视频路径
        choiceTime: 30,  // 视频播放到多少秒时显示选项
        choices: [
            { text: '选项文本', nextScene: '下一个场景ID' },
            // 更多选项...
        ]
    },
    // 更多场景...
};

你可以添加更多的场景和选项，创建更复杂的剧情分支结构。

技术说明

本项目使用以下技术：

  • HTML5：页面结构和视频播放
  • CSS3：样式和动画效果
  • JavaScript：游戏逻辑和交互控制

系统设计为响应式，可以在不同尺寸的设备上良好运行。
