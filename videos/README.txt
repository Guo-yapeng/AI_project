# 视频文件说明

请在此文件夹中放置以下视频文件，用于交互式剧情游戏：

1. intro.mp4 - 游戏介绍和初始场景
2. brave_path.mp4 - 选择勇敢前进后的场景
3. cautious_path.mp4 - 选择谨慎撤退后的场景
4. explore_alone.mp4 - 独自探索结局
5. seek_help.mp4 - 寻求帮助结局
6. gather_info.mp4 - 收集信息结局
7. make_plan.mp4 - 制定计划结局

## 视频要求

- 格式：MP4（推荐）
- 分辨率：建议720p或1080p
- 时长：根据剧情需要，通常每个场景1-3分钟

## 提示

如果您暂时没有合适的视频，可以使用占位视频进行测试。您可以从以下途径获取测试视频：

1. 使用免费的视频素材网站下载样本视频
2. 使用简单的屏幕录制工具创建带有文字说明的视频
3. 使用在线视频生成工具创建简单的动画视频

重要：请确保您使用的视频没有版权问题，或者您已获得使用许可。
